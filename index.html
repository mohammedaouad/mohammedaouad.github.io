Projects

01
AutoRegCT
SimpleITK TotalSegmentator Streamlit CT Registration
+
1 / 3 Streamlit dashboard
2 / 3 3D registration overlay
3 / 3 Metal artifact test case

Automated pre-op to post-op CT registration for total hip arthroplasty. The pipeline segments the femur from both scans, rigidly registers pre-op into post-op space, and computes 7 implant positioning metrics. No manual landmark placement, no visual alignment. What used to take 25 minutes per patient runs in under 2 minutes.

Registration uses Mattes mutual information with a 3-level multi-resolution pyramid. Validated on 4 synthetic test cases including a metal artifact simulation: the Mattes MI objective handled intensity corruption from beam hardening with a Dice of 0.87 and rotation recovery within 0.44 degrees.

Metrics computed 7 (Dice, axis angle, centroid offset, leg length...)
Validation 4 synthetic THA cases incl. metal artifact
Interface Streamlit GUI + CLI + JSON export

DICOMVault
pydicom SimpleITK PyQt6 Anonymization
+
1 / 3 MPR viewer
2 / 3 Anonymization checklist
3 / 3 Segmentation overlay

A desktop DICOM viewer and anonymizer built for research workflows. Loads a CT series, shows it in axial, coronal, and sagittal planes with synchronized crosshair navigation, then lists every patient-identifiable tag found in the file. You see exactly what's there before you decide what to strip.

De-identification covers 21 tags. UIDs are regenerated on every run so the output has no traceable link to the source. Also supports TotalSegmentator auto-segmentation and RT-Struct overlay, with export to anonymized DICOM or a 9-panel PNG report.

Tags de-identified 21 patient-identifiable fields + UID regeneration
Segmentation TotalSegmentator auto + RT-Struct manual load
Export Anonymized DICOM series + 9-panel PNG report

03
StemVis
PyVista PyQt6 SciPy Surgical Planning
+
1 / 3 Dual viewport GUI
2 / 3 HJC sphere fit
3 / 3 Neck resection plane

3D surgical planning tool for total hip arthroplasty. Takes a femur STL and automatically computes the hip joint center via sphere fitting, the shaft axis via PCA and SVD, and the stem tip position from the CCD angle. A dual-viewport GUI shows the pre-resection and post-resection femur side by side.

CCD angle and neck offset are adjustable via sliders, with live 3D feedback. The full pipeline runs in under 2 seconds on CPU. Works end-to-end from raw DICOM through a CT-to-STL pipeline using TotalSegmentator.

HJC detection Sphere fit, radius ~22-25 mm (anatomically consistent)
Planning controls CCD angle (115-135 degrees), neck offset (+-20 mm)
Pipeline DICOM to STL to 3D plan, fully automated

Brain Tumor Segmentation
PyTorch U-Net BraTS 2020 Grad-CAM
+
1 / 3 4-panel prediction figure
2 / 3 Grad-CAM heatmap
3 / 3 Training loss curve

Binary brain tumor segmentation on BraTS 2020 using a 3-level U-Net trained from scratch. Every component written manually: data loading from HDF5, model architecture, Dice loss, training loop, inference, and Grad-CAM visualization via PyTorch hooks on the bottleneck layer.

Trained on Google Colab T4 GPU across 5 epochs on 57,195 axial slices. The output shows FLAIR input, predicted mask with tumor volume estimate, ground truth, and a Grad-CAM heatmap showing which image regions drove the prediction.

Dice score 0.937 on sample slice
Training data 57,195 slices, 369 patients (BraTS 2020)
Explainability Grad-CAM via manual PyTorch hooks

05
Blood Pump PID Control
SciPy System ID PID Control Theory
+
1 / 3 Step response
2 / 3 Bode plot
3 / 3 Real-time tracking dashboard

System identification and PID controller design for an implantable centrifugal blood pump. Models three subsystems (MDU, motor/pump, flow sensor) using PT1 and PT2 transfer functions fitted to step response data via SciPy curve fitting.

The PID uses Ziegler-Nichols auto-tuning with a 35% Kp de-rating to reduce overshoot, and includes anti-windup via integral reversal on saturation. A real-time matplotlib dashboard shows the step response, tracking, Bode plot, and bandwidth sweep.

Settling time ~30 s, less than 5% overshoot
Tuned gains Kp 2.06, Ti 3.41 s, Td 0.85 s
Control range 2-6 L/min flow rate tracking
